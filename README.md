# Тикет-система — REST API на Go

Простая система учёта тикетов (заявок) с реализацией REST API и CRUD-операций. Написана на Go без внешних веб-фреймворков.

## О проекте

Учебный проект для практики построения REST API на чистом Go: создание, чтение, обновление и удаление тикетов через HTTP-эндпоинты.

В репозитории два файла:
- **main1.go** — черновой рабочий шаблон, первая версия с базовой логикой;
- **main2.go** — доработанная, более продакшн-готовая версия с улучшенной структурой кода.

## Технологии

- **Backend:** Go (net/http)
- **Архитектура:** REST API, CRUD

## Цель проекта

Закрепить понимание построения REST API на Go «с нуля»: маршрутизация запросов, обработка HTTP-методов, структурирование CRUD-логики.
